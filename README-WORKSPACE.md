# Cafe BrewsCo Website

Monorepo for the Cafe BrewsCo web projects.

## Overview

- `stay-awhile-cafe/` — The main website (Vite + React + TypeScript + Tailwind).
- `chatgpt-export/` — Export/demo bundle related to ChatGPT experiments.
- `chatgpt-eval-bundle/` — Evaluation bundle and notes for ChatGPT-related code.

## Quickstart

Requirements: Node.js (recommended v18+), npm.

To run the main site locally:

```powershell
cd stay-awhile-cafe
npm install
npm run dev
```

Useful scripts in `stay-awhile-cafe`:

- `npm run dev` — Start Vite dev server
- `npm run build` — Build production assets
- `npm run preview` — Preview the production build
- `npm run lint` — Run ESLint

## Repository structure

- `stay-awhile-cafe/` — Main site source (React + routes + components)
- `chatgpt-export/`, `chatgpt-eval-bundle/` — experimental/export bundles
- `package.json` — Root package file (if present)
